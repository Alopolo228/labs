# Задание 00  
Напишите две функции для решения задач своего варианта - с использованием рекурсии и без. Не используйте глобальные переменные и прочие средства хранения состояния между вызовами. Функция, которая разделяет список на n частей.  
# Описание  
1. Создаём функцию split со списком и количеством подсписков на вход  
2. Если количество подсписков равно или меньше нуля, возвращаем пустой список  
3. Цикл для создания подсписков  
4. Создаём кортежи для каждого элемента с помощью enumerate  
5. С помощью деления с остатком распределяем элементы списка по подспискам  
6. Возвращаем ответ
