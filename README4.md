# Задание 00  
Замыкание, определяющее нахождение аргуметов в допустимых диапазонах.  
# Описание  
1. Создаём внешнюю функцию принимающую диапазон значений
2. Создаём декоратор
3. Создаём функцию принимающую значения для функции, которую приняли в пункте 2  
4. Распределяем принимаемые значения  
5. Создаём условие исходя из задачи 
6. Вывод ошибки если не соответстует диапазону  
8. Возвращаем ответ если правильно

# Задание 01  
Декоратор, который будет оборачивать каждую функцию в try блок для обработки ошибок.  
# Описание  
1. Создаём декоратор    
2. Создаём функцию принимающую значения    
3. Возвращаем вывод функции если ошибок нет   
4. Возвращаем имя функции и ошибку если ошибка
