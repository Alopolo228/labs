# Задание  
Составить из словаря координат городов словарь расстояний между ними
# Описание  
1. Создать пустой словарь для хранения результатов
2. Для каждого города создать вложенный словарь
3. Для каждого города вычислить расстояние до первого города
4. Если города совпадают - расстояние равно 0
5. Если города разные - вычислить расстояние по формуле евклидовой метрики
6. Сохранить результат во вложенной словарь
