# Задание  
Генератор, который обращается к внешнему API и возвращает результаты запросов. Накопите результаты за определённый промежуток времени и сохраните в файл.
# Описание  
1. Импортируем библиотеки
2. Создаём генератор рандомных чисел
3. Переменную времени с начала эпохи  
4. Переменную с адресом API  
5. Создаём переменную запроса 
6. Случаи с возращённым ответом и ошибкой  
7. Создаём генератор записи результатов в файл
8. Переменную времени с начала эпохи  
9. Переменную времени с начала работы генератора
10. Переменную результата работы генератора  
11. Записываем в файл  
12. Функцию для перебора значений переменной  
13. Запускаем генератор  
14. Выводим результаты
15. В список записываем результаты работы  
16. Создаём список только с числами получившимися в ходе запросов  
17. Выводим их
