Задание 00

Составить из словаря координат городов словарь расстояний между ними
Описание

    Создать пустой словарь для хранения результатов
    Для каждого города создать вложенный словарь
    Для каждого города вычислить расстояние до первого города
    Если города совпадают - расстояние равно 0
    Если города разные - вычислить расстояние по формуле евклидовой метрики
    Сохранить результат во вложенной словарь

Задание 01

Есть значение радиуса круга, выведите на консоль значение площади этого круга с точностью до 4-x знаков после запятой
Описание

    Создана переменная равная значению пи
    Вычислена площадь круга и округлена ло 4-х знаков после запятой
    Определено расстояние от точки до начала координат
    Если точка лежит внутри круга, то True, иначе False
    Аналогично для второй точки
